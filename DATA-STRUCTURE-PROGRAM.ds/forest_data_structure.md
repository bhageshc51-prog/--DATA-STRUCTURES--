# Forest

## Definition

A Forest is a collection of disjoint trees in Data Structure.

If a tree is removed from its root node, the remaining subtrees form a forest.

---

## Properties

1. A forest contains multiple trees.
2. Each tree is independent.
3. There are no cycles in a forest.
4. Forest can be converted into a tree by adding a root node.

---

## Example

Tree 1:
A → B, C

Tree 2:
D → E, F

Together they form a Forest.

---

## Applications

- File Systems
- Database Systems
- Hierarchical Data Storage
- Network Structures

---

## Conclusion

Forest is used to represent multiple hierarchical structures together in Data Structures.
